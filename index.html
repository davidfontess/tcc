Introdução

Bem vindo ao Giz!

O que é Giz?

Giz é uma plataforma web que ensina programação funcional de forma visual, interativa e divertida. A linguagem de programação do Giz é JavaScript, no entanto, programas Giz são funcionais puros, pois apenas utilizam a faceta funcional de JavaScript.

Combinando elementos de um ambiente de desenvolvimento com uma interface, os usuários podem aprender a programar criando imagens, animações e programas funcionais reativos como, por exemplo, jogos. Os programas funcionais podem ser executads imediatamente após escritos apenas apertando um botão.

O Giz foi projetado especialmente para crianças e adolescentes, por isso o processo de aprendizado é lúdico e visual. Seu principal objetivo é ensinar conceitos de programação funcional por meio de programas que manipulam figuras e criam animações. Ao invés de focar em estruturas e sintaxe complexas e palavras em inglês, este guia tenta apresentar os códigos mais próximos da matemática básica ensinada nas escolas, assim facilitando a introdução de programação e possibilitando o uso de código como uma forma de expressão criativa.

O ambiente Giz

Ao fechar este guia, à esquerda, você verá o editor de código, onde você digita seus comandos. À direita, temos o canvas, onde os desenhos criados pelo seu código irão aparecer.

O botão Executar excuta o programa escrito no editor e, como consequência, apresenta direto no canvas a imagem especificada no programa .

O botão Parar interrompe a execução do programa, apagando a imagem do canvas.

Os botões A+ e A- servem para aumentar ou diminuir o tamanho da fonte (as letras que você escreve) do editor.

O botão Claro/Escuro muda esta página entre o modo claro e o modo escuro, para melhor conforto visual.

O botão Abrir Guia/Fechar Guia abre e fecha este guia, onde você vai encontrar explicações de como fazer tudo que o Giz pode fazer.

Dentro do guia, você pode clicar nos títulos do sumário para abrir cada parte de conteúdo.

Cada seção possui exemplos de código que podem ser clicados. Ao clicar, o código é automaticamente colado no editor, pronto para ser executado!

Para fechar o guia, basta clicar no botão verde "Fechar Guia" no canto superior direito da tela.

Parte 1: Primeiros Passos – Figuras do Giz

Receita dos programas

Com relação a figuras, no Giz podemos fazer duas coisas principais: criar figuras e desenhar figuras. No começo pode parecer que essas duas ações são a mesma coisa, mas não são!

Imagine que o Giz é como um desenho animado ou uma peça de teatro. Você pode preparar um personagem, imaginar e descrever suas roupas, o jeito como ele fala, como se move... Mas ele só vai aparecer no palco quando o diretor disser: "Ação!"

No Giz é igual: criar uma figura é como preparar o personagem. Desenhar uma figura é colocá-lo no palco, ou melhor, na tela.

Então, lembre-se: só porque você criou uma figura, não quer dizer que ela vai aparecer. Para isso, você precisa dizer ao Giz para desenhá-la.

Vamos começar aprendendo a criar figuras!

Figuras Básicas

Antes de partirmos para figuras mais complexas, vamos começar com algo simples. Existem três figuras que chamamos de básicas, porque precisam de pouco código para aparecer. Por isso, são ótimas para começar.

Círculo

Vamos começar com o exemplo mais simples possível: um círculo.

Para desenhar um círculo, precisamos de apenas duas linhas de código: uma para criar o círculo, e outra para desenhá-lo na tela (também chamada de canvas).

const meuCirculo = circulo(2);
desenhar(meuCirculo);

Explicação:

Na primeira linha, escrevemos const meuCirculo =. A palavra const sempre será usada ao criar figuras (vamos entender mais sobre ela em breve).

meuCirculo é o nome que demos para a figura. Você pode escolher o nome que quiser: circ, circulo1, bola, sol... Você quem decide!

circulo(2) quer dizer que estamos criando um círculo com raio 2. (Logo você entenderá o que significa essa unidade de medida.)

Resumindo a primeira linha: const meuCirculo = circulo(2); significa algo como: "meuCirculo será um círculo com raio de 2 unidades."

Perceba que usamos const no início e ; no final. Essas partes ainda não têm tradução, mas são obrigatórias na criação de figuras.

Na segunda linha, temos: desenhar(meuCirculo);. Isso quer dizer: "Desenhe a figura meuCirculo."

O comando desenhar sempre será usado uma única vez, no fim do código. É como o momento em que o diretor grita: "Ação!"

Útil: existe uma forma de escrever coisas no código que serão ignoradas pelo nosso programa. Pode até parecer inútil, mas e se você quiser fazer alguma anotação para lembrar depois?

Quando começamos uma linha com // aquela linha inteira é ignorada pelo programa. Chamamos isso de comentário. Tente adicionar um comentário explicando que esse código é um círculo.

Retângulo

Agora vamos criar um retângulo. Assim como o círculo, precisamos de duas linhas: uma para criar e outra para desenhar.

const meuRetangulo = retangulo(2, 4);
desenhar(meuRetangulo);

Explicação:

Na primeira linha, usamos const meuRetangulo = para dar um nome à figura que vamos criar.

O nome escolhido aqui foi meuRetangulo, mas você pode usar qualquer nome que quiser, como ret, janela, tijolo ou retangulo1.

Em seguida, temos retangulo(2, 4). Isso quer dizer que queremos um retângulo com largura 2 e altura 4. (Depois vamos entender melhor o que essas medidas significam e como o tamanho aparece na tela. Não se preocupe com isso agora.)

Resumindo, essa primeira linha cria um retângulo com tamanho 2 por 4 e dá a ele o nome meuRetangulo.

Na segunda linha, temos desenhar(meuRetangulo);, que é o comando que realmente coloca a figura na tela. Ou seja: "Desenhe o meu retângulo!"

Lembre-se: você não pode só criar a figura, precisa mandá-la aparecer com o comando desenhar(figura), como se fosse o momento de brilhar no palco!

Plano Cartesiano

Algumas figuras já estão prontas no Giz. Uma delas é o planoCartesiano. Ele nos ajuda a entender melhor onde os desenhos estão sendo feitos, nos mostrando pontos (x,y) específicos.

Para desenhar o plano cartesiano, não precisamos criar nada antes. Basta usar:

desenhar(planoCartesiano);

Explicação:

Como o planoCartesiano já existe, podemos desenhá-lo diretamente. Esse comando mostra na tela as linhas do plano e o ponto central, que é o ponto (0, 0). Ele ajuda bastante quando queremos saber exatamente onde nossas figuras estão sendo desenhadas.

Agora, se quisermos desenhar uma figura nossa *junto* com o plano cartesiano, podemos desenhar os dois ao mesmo tempo.

const circ = circulo(2);
desenhar([circ, planoCartesiano]);

Explicação:

Aqui, criamos um círculo chamado circ e depois pedimos para o Giz desenhar duas coisas ao mesmo tempo: o circ e o planoCartesiano.

Quando usamos colchetes [], estamos dizendo ao Giz: "Desenhe esses elementos juntos". Dessa forma, conseguimos ver o plano cartesiano e a figura que desenhamos/estamos desenhando. Assim fica mais fácil saber onde é um ponto (x,y).

Você pode colocar quantas figuras quiser dentro dos colchetes, separadas por vírgula. Assim o Giz desenha todas elas de uma vez. Só tenha em mente que a ordem dessas figuras vai importar. Figuras que estão mais à esquerda em desenhar() ficarão mais atrás na nossa tela.

Figuras um pouquinho mais complexas

Agora que já conhecemos as figuras básicas como o círculo, o retângulo e o plano cartesiano, vamos aprender como criar figuras um pouco mais avançadas. Essas figuras precisam de mais informações para serem desenhadas, mas não se preocupe, vamos aprender passo a passo!

Vamos começar com a linha.

Desenhando linhas

Outra figura um pouquinho mais complexa, mas muito útil, é a linha. Com ela podemos ligar dois pontos no plano, como se estivéssemos usando uma régua.

Vamos ver um exemplo:

const minhaLinha = linha(-5,-5,5,5);
desenhar(minhaLinha);

Explicação:

No comando linha, precisamos dizer de onde até onde queremos que a linha vá. Os quatro números representam dois pontos:

  • -5, -5: esse é o ponto inicial da linha (mais para a esquerda e para baixo);
  • 5, 5: esse é o ponto final da linha (mais para a direita e para cima).

Ou seja, estamos ligando o ponto (-5, -5) até o ponto (5, 5).

Depois, usamos desenhar(minhaLinha); para mostrar a linha na tela.

Você pode criar linhas na horizontal, na vertical, na diagonal... é só mudar os valores dos pontos extremos! Linhas são ótimas para criar grades, contornos, e até formar desenhos mais elaborados.

Exercício: Desenhando uma cruz e um X

Vamos praticar um pouco mais o uso das linhas. Tente fazer os seguintes desafios:

Desafio 1: Uma cruz (+)

Dica: uma cruz é formada por uma linha vertical e outra horizontal que se cruzam no centro.

Exemplo de solução:

const linha1 = linha(-5, 0, 5, 0);
const linha2 = linha(0, -5, 0, 5);
desenhar([linha1, linha2]);
Desafio 2: Um X

Dica: um X é feito com duas linhas diagonais que se cruzam no centro da tela.

Exemplo de solução:

const linha1 = linha(-5, -5, 5, 5);
const linha2 = linha(-5, 5, 5, -5);
desenhar([linha1, linha2]);

Agora é sua vez! Experimente mudar os números para ver o que acontece. Tente criar outros formatos, como um quadrado usando quatro linhas!

Polilinha

A polilinha é como uma figura feita com vários segmentos de linha conectados, formando um caminho. É muito útil quando queremos desenhar algo que não seja só uma linha reta, como uma montanha, uma seta ou até parte de um desenho mais elaborado.

Veja este exemplo:

const minhaPolilinha = polilinha([[0, 0],[2, 3],[4, 0],[2, 0]]);
desenhar(minhaPolilinha);

Explicando:

  • A função polilinha([...]) recebe uma lista de pontos. Cada ponto é representado por dois números: a posição X e a posição Y.
  • No nosso exemplo, temos quatro pontos: [0,0], [2,3], [4,0] e [2,0].
  • O Giz vai ligar todos esses pontos com linhas, uma depois da outra, formando um caminho.

Podemos imaginar que estamos "desenhando com o dedo", ligando os pontos em ordem. O resultado parece uma pequena montanha com uma base.

Você pode usar quantos pontos quiser, e até combinar várias polilinhas para criar formas mais elaboradas. Experimente mudar os pontos ou adicionar mais para ver o que acontece!

Polígono

O polígono é uma figura geométrica fechada, formada pro segmentos de reta. Um polígono pode ter quantos lados você quiser. Pode ter 3 (ou seja um triângulo), 4 pontos (um retângulo ou quadrado), ou mais. Para desenhá-lo, usamos algo a palavra poligono. Usando ela, precisamos dizer os pontos que formam os cantos do losango.

Veja o exemplo:

const losangulo = colorir(poligono([[-7,0],[0,5],[7,0],[0,-5]]),"amarelo");
desenhar(losangulo);

Explicação:

Primeiro usamos o comando poligono e passamos dentro dos parênteses uma lista com quatro pontos. Cada ponto é escrito entre colchetes com dois números: o primeiro número indica a posição horizontal (para a direita ou esquerda) e o segundo indica a posição vertical (para cima ou para baixo).

Os pontos que usamos aqui são:

  • [-7, 0]: um ponto mais à esquerda
  • [0, 5]: um ponto no alto, no centro
  • [7, 0]: um ponto mais à direita
  • [0, -5]: um ponto mais abaixo

Esses quatro pontos juntos formam o losango.

Por fim, usamos o desenhar(...) para mostrar o losango na tela.

Mesmo sendo um pouco mais trabalhoso, criar figuras com poligono nos dá muito mais liberdade para desenhar o que quisermos! Um triângulo, um quadrilátero, um pentágono, um hexágono, e assim por diante.

Texto

O Giz também permite desenhar palavras e frases! Para isso, usamos a função texto.

Veja este exemplo:

const meuTexto = texto("Olá, GIZ!", "24px", "vermelho");
desenhar(meuTexto);

Explicando:

  • texto("Olá, GIZ!", "24px", "vermelho") cria uma figura com o texto Olá, GIZ!, escrito com tamanho de letra 24 pixels e na cor vermelha.
  • Você pode trocar o que está escrito, o tamanho da letra (ex: "20px", "30px") e a cor (ex: "azul", "verde", "preto").

Assim como nas outras figuras, primeiro criamos o texto, depois usamos desenhar para mostrá-lo na tela.

Experimente mudar a frase, o tamanho e a cor para criar seus próprios títulos e mensagens! Pode, por exemplo, escrever seu nome!

Parte 2: Programação Funcional

O que é Programação Funcional?

O nome pode parecer complicado à primeira vista, mas na verdade é mais simples do que parece.

O ponto principal da programação funcional é focar no que deve ser feito. Para isso, programamos sequências de funções que tranformam dados. Na programação funcional possuimos funções puras, ou seja, fornecendo a mesma entrada, sempre teremos o mesmo resultado. Dessa forma, baseado na entrada, conseguimos prever a execução do código e evitar problemas. Além disso, após serem criadas, variáveis, funções e dados em geral não podem ser modificados. Chamamos esse conceito de Imutabilidade. Ademais, funções porem ser tratadas como um valor como qualquer outro e até mesmo usadas como argumento ou a saída de outras funções.

Funções: os blocos mágicos

Uma função é como uma máquina. Você coloca uma coisa nela, e ela devolve outra. Por exemplo:

const dobro = x => x * 2;
desenhar(texto(dobro(4)));

Nesse exemplo, criamos uma função chamada dobro. Ela recebe um número (x) e devolve esse número multiplicado por 2.

Quando escrevemos dobro(4), o resultado é 8.

Reutilizando funções

O legal de criar funções é que você pode usá-las várias vezes, sempre que quiser. Por exemplo:

const metade = x => x / 2;
const raio = 8;
const bola = circulo(metade(raio));
const bolaMenor = colorir(circulo(metade(metade(raio))),"azul");
const duasBolas = comporFiguras([bola, bolaMenor]);
desenhar(duasBolas);

A função metade pega um número e devolve a metade dele. No exemplo, usamos ela para criar um círculo com raio 2 (porque a metade de 4 é 2). Depois, se você precisar, pode chamar metade de novo para fazer essa operação com outros valores e variáveis.

Funções com mais de uma entrada

Você também pode criar funções que recebem mais de um valor. Veja só:

const desenharCarro = () => {
  const capo = mover(colorir(retangulo(3,2),"azul"),0,1);
  const base = colorir(retangulo(6,2),"azul");
  const roda = mover(circulo(1),-1.5,-1.5);
  const roda2 = mover(circulo(1),1.5,-1.5);
  const carro = comporFiguras([capo,base,roda,roda2]); 
  return carro;
};

const conjuntoDeCarros = comporFiguras([
	desenharCarro(),
    mover(desenharCarro(),0,5),
    mover(desenharCarro(),0,-5),
    mover(desenharCarro(),-7,0),
    mover(desenharCarro(),7,0),
    mover(desenharCarro(),-7,5),
    mover(desenharCarro(),-7,-5),
    mover(desenharCarro(),7,5),
    mover(desenharCarro(),7,-5)
]);

desenhar(conjuntoDeCarros);

Aqui criamos uma função chamada retanguloPersonalizado que recebe duas entradas: l (largura) e a (altura).

Depois usamos essa função para criar um retângulo com largura 3 e altura 5. Percebe que você consegue criar suas próprias formas usando outras formas já pré-existentes no Giz?

Rotacionando figuras

Com a função rotacionar, você pode girar uma figura em torno do seu centro.

const ret = retangulo(4, 2);
const retRotacionado = rotacionar(ret, 45);
desenhar(retRotacionado);

Isso é útil para fazer hélices, setas apontando em direções diferentes ou criar efeitos visuais giratórios.

Redimensionando figuras

Quer aumentar ou diminuir uma figura? Use a função redimensionar!

const ret = retangulo(4, 2);
const retRedimensionado = redimensionar(ret, 2);
desenhar(retRedimensionado);

Nesse exemplo, multiplicamos o tamanho do retângulo por 2. Ele fica com o dobro da largura e da altura.

Colorindo figuras

Você também pode escolher a cor da sua figura com a função colorir. É só dizer qual cor quer!

const ret = retangulo(4, 2);
const retColorido = colorir(ret, "verde");
desenhar(retColorido);

Agora seu desenho pode ganhar vida com cores diferentes. Você pode usar nomes como "vermelho", "azul", "amarelo" e muitos outros.

Compondo figuras

Você pode juntar várias figuras para formar uma composição mais rica. É como montar um desenho com várias peças!

const ret = retangulo(4, 2);
const figurasCompostas = comporFiguras([ret, circulo(3), poligono([[0,0],[2,3],[4,0]])]);
desenhar(figurasCompostas);

No exemplo acima, criamos um retângulo, um círculo e um polígono e os juntamos em uma única imagem com comporFiguras.

É como desenhar várias partes de um robô: cabeça, corpo, braços... tudo se junta!

Combinando funções com figuras

Funções são ainda mais legais quando usamos elas para transformar figuras! Veja esse exemplo:

const bola = circulo(3);
const moverParaDireita = fig => mover(fig, 5, 0);
const novaBola = moverParaDireita(bola);
desenhar([bola, novaBola]);

Nesse exemplo, criamos uma bola e depois usamos uma função para mover ela 5 unidades para a direita.

Com isso, temos duas figuras: a original e a nova, movida. E tudo isso graças a uma função!

Explorando possibilidades

Você pode combinar todas essas transformações para criar desenhos incríveis. Experimente girar uma figura colorida, redimensionar uma composição inteira ou mover tudo para outra posição!

Resumo

  • Programação funcional usa funções para transformar dados (ou figuras!)
  • Funções podem ter uma ou mais entradas
  • Você pode usar funções para deixar seu código mais organizado e poderoso

Parte 3: Animações – Dando Vida às Figuras

Movimento no Giz

Até agora, criamos figuras que ficam paradas na tela. Mas e se quisermos que elas se movimentem? Que tal fazer um círculo andar de um lado para o outro, ou um retângulo subir e descer?

No Giz, podemos criar animações usando uma função especial chamada animar.

O segredo está em uma palavra nova: tempo ou t. Essa letrinha representa o tempo que passou desde o início da animação. Conforme o tempo passa, o valor de t vai aumentando. E é isso que usamos para movimentar as figuras!

Exemplo básico: um círculo que anda

const circ = circulo(2);
animar((tempo) => {
	return mover(circ, tempo, 0);
});

Explicação:

Vamos entender esse código passo a passo:

  • animar(...) inicia a animação.
  • Usamos (tempo) => { ... } para dizer o que acontece a cada momento.
  • Dentro da função, criamos uma figura com circulo(2) e usamos mover(circ, tempo, 0) para movê-la no eixo X.
  • O return indica qual figura deve ser desenhada a cada instante.

O resultado é um círculo deslizando para a direita com o tempo!

Movimento em outras direções

Para fazer a figura subir, troque o eixo X pelo Y:

const circ = circulo(2);
animar((tempo) => {
	return mover(circ, 0, tempo);
});

Para andar na diagonal, mova nos dois eixos ao mesmo tempo:

const circ = circulo(2);
animar((tempo) => {
	return mover(circ, tempo, tempo);
});

Outros movimentos

Com contas envolvendo tempo, você pode criar movimentos interessantes. Exemplo com onda:

const circ = circulo(2);
animar((tempo) => {
	return mover(circ, Math.sin(tempo), 0);
});

Movimento circular combinando seno e cosseno:

const circ = circulo(2);
animar((tempo) => {
	return mover(circ, Math.cos(tempo) * 5, Math.sin(tempo) * 5);
});

A figura gira em uma trajetória circular com raio 5.

Agora é sua vez! Experimente criar uma cena com personagens se movendo!

Parte 4: Interatividade – Respondendo ao Usuário

Vamos deixar tudo mais interativo!

Você já aprendeu a criar figuras e até animar elas com o tempo. Agora chegou a hora de dar mais um passo: fazer o programa responder ao que o usuário faz!

Com a função interagir, seu programa pode escutar o que a pessoa está fazendo, como mover o mouse, clicar ou apertar uma tecla, e mudar o que está desenhando em tempo real!

Conhecendo o "estado"

Para lidar com interações, precisamos guardar informações. Por exemplo, onde está o personagem? O que ele está fazendo?

Essas informações ficam em algo que chamamos de estado. O estado é um conjunto de dados que seu programa pode usar, mudar e desenhar.

Um exemplo simples: seguindo o mouse

Vamos ver um exemplo onde um círculo segue a posição do mouse:

// Círculo segue o mouse
const mundoInicial = { x: 0, y: 0 };

const idTempo = (t, mundo) => mundo;

const mudEvt = (evt, mundo) => {
  if (evt.tipo === "movimento") {
    return { x: evt.posicao[0], y: evt.posicao[1] };
  }
  return mundo;
};

const apresentaMundo = (mundo) => {
  return mover(circulo(1), mundo.x, mundo.y);
};

interagir(mundoInicial, idTempo, mudEvt, apresentaMundo);

Exemplo com clique: mudar de cor

Vamos fazer um círculo que muda de cor a cada clique:

// Círculo muda de cor ao clicar
const mundoInicial = { 
    cor: "vermelho" 
};

// função que atualiza o estado com o tempo (não faz nada neste exemplo)
const idTempo = (tempo, mundo) => {
    return mundo; // Mantém o estado inalterado
};

// função que trata os eventos do usuário
const mudEvt = (evento, mundo) => {
    // verifica se foi um clique do mouse (botão esquerdo)
    if (evento === "clickLeft") {
        // alterna entre vermelho e azul
        return {
            cor: mundo.cor === "vermelho" ? "azul" : "vermelho"
        };
    }
    return mundo; // retorna o estado inalterado para outros eventos
};

// função que converte o estado em figura para desenhar
const apresentaMundo = (mundo) => {

Usando o teclado

Você pode usar as teclas para controlar uma figura. Veja esse exemplo:

// controlando com as setas do teclado
const meuRetangulo = retangulo(2, 1, 0, 0);
const mundoInicial = { x: 0, y: 0 };
const idTempo = (t, mundo) => mundo;
const mudEvt = (evt, mundo) => {
	switch (evt) {
		case "ArrowRight":
			return { ...mundo, x: mundo.x + 1 };
		case "ArrowLeft":
			return { ...mundo, x: mundo.x - 1 };
		case "ArrowUp":
			return { ...mundo, y: mundo.y + 1 };
		case "ArrowDown":
			return { ...mundo, y: mundo.y - 1 };
		default:
			return mundo;
	}
};
const apresentaMundo = (mundo) => {
	return mover(retangulo(3, 1), mundo.x, mundo.y);
};
interagir(mundoInicial, idTempo, mudEvt, apresentaMundo);

Use as setas do teclado para mover o círculo pela tela!

Resumo

  • interagir permite criar programas que respondem a ações do usuário
  • Você usa um estado para guardar informações
  • Eventos como movimento, clique e tecla podem mudar o estado
  • Com isso, seus programas ficam vivos e interativos!
